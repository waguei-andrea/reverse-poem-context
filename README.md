## How to use

```sh
git clone https://github.com/waguei-andrea/reverse-poem-context.git
cd reverse-poem
```

install the packages this project need:
```sh
yarn install
```
after everything is installed, then:
```sh
yarn start
```

then, browse http://localhost:3000/

There it is! Enjoy it!
